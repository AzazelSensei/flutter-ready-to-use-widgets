# WidgetSlider
İstediğiniz miktarda widget'ı istediğiniz hızda, dikey veya yatay olarak ekranda kaydırabilirsiniz.
Cache extent parametresinin ekran genişliğinde olmasıyla performans sıkıntısı yaşamayacaksınızdır.

## Çalışma Mantığı

WidgetSlider(
            slideAxis: ,
            slideCount: ,
            slidingSpeed: ,
            spacing: ,
            children: []
          ),
